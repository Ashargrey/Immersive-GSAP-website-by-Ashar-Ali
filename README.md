# 🌟 GSAP Interactive Animation Website

An immersive and visually engaging web experience built using **GSAP (GreenSock Animation Platform)**. This project focuses on advanced animation techniques, smooth transitions, and particle-based visual effects.

## 🚀 Live Demo
[Click here to view live](https://interactive-web-by-ashar-ali.netlify.app/)

---

## ✨ Features

- ⚡ Smooth GSAP powered animations
- 🎯 Interactive particle / fragment effects
- 🎨 Modern and clean UI design
- 💡 High-performance animation handling
- 🖱️ User interaction based transitions

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- GSAP (GreenSock Animation Platform)
